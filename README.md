# CryptoModern

**Acesso ao site:** [https://cryptomodern.netlify.app/](https://cryptomodern.netlify.app/)

---

## Descrição
CryptoModern é uma ferramenta web que permite **criptografar** e **descriptografar** arquivos usando uma **senha** definida pelo usuário. Ideal para proteger arquivos antes de enviar, armazenar em nuvem ou compartilhar com segurança.

---

## Funcionalidades
- **Criptografia de arquivos**: Proteja seus arquivos com uma senha única.
- **Descriptografia de arquivos**: Recupere seus arquivos utilizando a mesma senha.
- Interface simples e intuitiva.
- Funciona diretamente no navegador, sem necessidade de instalação.

---

## Como usar
1. Acesse o site: [https://cryptomodern.netlify.app/](https://cryptomodern.netlify.app/).  
2. Selecione o arquivo que deseja criptografar ou descriptografar.  
3. Insira a **senha** que será usada na criptografia ou que foi usada anteriormente.  
4. Clique no botão correspondente (Encrypt/Decrypt).  
5. Baixe o arquivo resultante.

---

## Segurança
- A segurança depende da **senha escolhida**: utilize senhas fortes para proteger seus arquivos.  
- CryptoModern não armazena arquivos nem senhas no servidor — todo o processamento é feito localmente no navegador.  

---

## Observações
- Não perca sua senha, pois sem ela não é possível recuperar o arquivo criptografado.  
- Recomendado para pequenos arquivos devido às limitações de processamento no navegador.
